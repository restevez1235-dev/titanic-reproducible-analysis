\# Reproducible Titanic Survival Analysis



A complete, end-to-end data science workflow using Python, Pandas, scikit-learn, and Jupyter.



\## Overview

Analyzes factors influencing survival on the Titanic using cleaning, EDA, and classification modeling.



\## Setup Instructions

1\. Clone: `git clone https://github.com/restevez1235-dev/titanic-reproducible-analysis.git`

2\. Install: `pip install -r requirements.txt`

3\. Run: Open `notebooks/titanic\_full\_analysis.ipynb` in Jupyter



\## Key Findings

\- Women and 1st-class passengers had dramatically higher survival rates

\- Engineered features (Title, Family\_Size) improved model performance

\- Logistic regression achieved ~82% accuracy and 0.87 AUC

\- data/: raw and processed
